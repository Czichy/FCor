FCor

Linear Algebra

FCor supports 5 matrix factorizations and related solvers:

  • Cholesky
  • LU
  • QR
  • Singular Value
  • Eigenvalue

You can also calculate matrix-vector products and transpose a matrix in/out of place.

1: 
2: 
3: 
4: 
5: 
6: 
7: 

#r "FCor.dll"

open FCor
open System
open FCor.ExplicitConversion
open FCor.Math
open FCor.LinearAlgebra

Matrix products and transposition

Use the operators * and ^* for calculating matrix-matrix, matrix-vector and vector-vector products:

1: 
2: 
3: 
4: 

let m1 : Matrix = !![[1.0; 2.0]
                     [3.0; 4.0]] 
let v1 : Vector = !![5.0; 6.0]
let m2 = m1 * m1

This gives a value for m2 of:

Size = [2,2]
   7  10
  15  22


1: 
2: 
3: 

let v2 = m1 * v1
let m3 = m1 ^* m1 // transpose(m1) * m1 without creating the transposed matrix
let prod = v1 * v1

v2 produces a value of:

Size = [2,2]
  10  14
  14  20

m3 produces a value of:

Size = [2,2]
 False False
  True False

prod produces a value of:

61.0

You can transpose a matrix in place or create a new matrix:

1: 

let m5 = transpose(m1)

m5 produces a value of:

Size = [2,2]
   1   3
   2   4


1: 

m1.Transpose()

m1 produces a value of:

Size = [2,2]
   1   3
   2   4

Cholesky factorization

Cholesky requires a square positive definite matrix. The factorization returns the upper tri matrix. You can also calculate the inverse matrix and solve A*x=b, where b can be a vector or a matrix:

1: 
2: 
3: 
4: 
5: 

let m6 : Matrix = !![[2.0;0.7;0.7]
                     [0.7;2.0;0.7]
                     [0.7;0.7;2.0]]
let b1 : Vector = !![1.0;2.0;3.0]
let m7 = chol m6

m7 produces a value of:

Size = [3,3]
   1.4142135623731 0.494974746830583 0.494974746830583
                 0  1.32476412994918 0.343457367023861
                 0                 0  1.27946748182087


1: 
2: 

let m8 = cholInv m6 // inverse
let v3 = cholSolve m6 b1

v3 produces a value of:

Length = 2
  17
  39

LU factorization

LU factorizes matrix A into lower and upper tri matrix: A = P*L*U where P is row permutation matrix. You can also calculate the inverse matrix and solve A*x=b, where b can be a vector or a matrix:

1: 

let l, u, p = lu m6

l produces a value of:

Size = [3,3]
                 1                 0                 0
              0.35                 1                 0
              0.35 0.259259259259259                 1

u produces a value of:

Size = [3,3]
                2              0.7              0.7
                0            1.755            0.455
                0                0 1.63703703703704

p produces a value of:

[|0; 1; 2|]
1: 
2: 

let m10 = luInv m6
let v4 = luSolve m6 b1

v4 produces a value of:

Length = 3
 -0.180995475113122
  0.588235294117647
   1.35746606334842

QR factorization

QR factorizes matrix A [mxn] into orthogonal Q and upper triangular R: A = Q*R. If m > n then economy factorization is performed so that Q has n columns and R has n rows. You can use it to solve linear least squares problems as well as compute effective rank for given numerical tolerance:

1: 

let m11 = new Matrix(4, 3, [2.0..13.0])

m11 produces a value of:

Size = [4,3]
   2   6  10
   3   7  11
   4   8  12
   5   9  13


1: 

let q, r = qr m11

q produces a value of:

Size = [4,3]
 -0.272165526975909 -0.791154805285239  0.473569510960267
 -0.408248290463863 -0.365148371670112 -0.426308272966961
 -0.544331053951817 0.0608580619450182 -0.568091986946873
 -0.680413817439771  0.486864495560149  0.520830748953568

r produces a value of:

Size = [3,3]
    -7.34846922834954     -14.969103983675    -22.5897387390004
                    0    -2.43432247780073    -4.86864495560147
                    0                    0 4.31132637123079E-15


1: 

let m12 = new Matrix(4, 2, [1.0..8.0])

m12 produces a value of:

Size = [4,2]
   1   5
   2   6
   3   7
   4   8


1: 
2: 

let m13 = qrSolveFull m11 m12 // assumes m11 has full rank
let m14, rank = qrSolve m11 m12 1e-20 // numerical tolerance is used to compute effective rank

m14 produces a value of:

Size = [3,2]
  0.975678733031674              0.125
  0.298642533936652                  1
 -0.274321266968326             -0.125

Singular value factorization

Singular value factorization decomposes matrix A [mxn] into diagonal S and unitary U and V: A = U * S * V.T. If m > n then economy factorization is performed so that U has n columns. You can use it to solve linear least squares problems as well as compute singular values:

1: 
2: 
3: 

let m15 : Matrix = !![[1.0;3.0]
                      [2.0;4.0]
                      [3.0;5.0]]

m15 produces a value of:

Size = [3,2]
   1   3
   2   4
   3   5


1: 

let U, S, Vt = svd m6 //S is vector of singular values, vt = transpose(V)

U produces a value of:

Size = [3,3]
    -0.577350269189626     0.816496580927726 -2.63153586407333E-17
    -0.577350269189626    -0.408248290463863    -0.707106781186547
    -0.577350269189626    -0.408248290463863     0.707106781186548

S produces a value of:

Length = 3
 3.4
 1.3
 1.3

Vt produces a value of:

Size = [3,3]
 -0.577350269189626 -0.577350269189626 -0.577350269189626
  0.816496580927726 -0.408248290463863 -0.408248290463863
                  0 -0.707106781186547  0.707106781186548


1: 

let m16, rnk = svdSolve m6 m15 1e-20

m16 produces a value of:

Size = [3,2]
 -0.180995475113122  0.407239819004525
  0.588235294117647   1.17647058823529
   1.35746606334842   1.94570135746606


1: 

let singVals = svdValues m6 // singular values

singVals produces a value of:

Length = 3
 3.4
 1.3
 1.3

Eigenvalue factorization

Eigenvalue factorization decomposes matrix A [nxn] into eigenvectors V and eigenvalues D so that A * V = V * D

1: 
2: 
3: 

let m17 : Matrix = !![[1.0;3.0;5.0]
                      [3.0;2.0;4.0]
                      [5.0;4.0;1.0]]

m17 produces a value of:

Size = [3,3]
   1   3   5
   3   2   4
   5   4   1


1: 

let m18, v5 = eig m17

m18 produces a value of:

Size = [3,3]
  -0.62749998092142  0.536439079440068  0.564337565640557
 -0.186108977893998 -0.807122434366506  0.560282807419197
  0.756047103222325  0.246549163437595  0.606305440943221

v5 produces a value of:

Length = 3
 -0.180995475113122
  0.588235294117647
   1.35746606334842
